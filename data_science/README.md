# Data Science

Bu klasor VentureAgent'in veri bilimi calismalari icindir.

## Klasorler

- `datasets/`: Ham veya islenmis veri dosyalari. Buyuk dosyalari Git'e eklemeyin.
- `notebooks/`: Kesif, analiz ve deney notebooklari.
- `models/`: Egitilmis model ciktilari veya prototip model dosyalari.
- `pipelines/`: Uygulamada tekrar kullanilabilecek Python analiz kodlari.

## Ilk hedefler

1. Girisim fikri icin pazar hazirlik skoru uretmek.
2. SWOT, rakip ve gelir modeli sonuclarindan sayisal ozellikler cikarmak.
3. Kullanici cevaplarini daha sonra model egitimi icin anonim veri setine donusturmek.
